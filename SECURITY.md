# Security Policy

## Supported Versions

| Version  | Supported |
| -------- | --------- |
| >= 0.1.0 | yes       |

## Reporting a vulnerability

If you believe you’ve found a security issue, please **do not** open a public GitHub issue.

Instead, use one of the following:

- GitHub Security Advisories (preferred)
- Email: `fabian@shiftbloom.studio`

## Secrets

- Never commit tokens/keys/passwords.
- If a token is accidentally shared, **revoke/rotate it immediately**.
