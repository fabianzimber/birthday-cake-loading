# Birthday-Cake Loading — Nuxt demo

Minimal Nuxt demo for `@birthday-cake-loading/nuxt`.

## Run

```bash
npm install
npm run dev
```

The module automatically:
- sets **Client Hints** headers (Accept-CH / Permissions-Policy / Vary)
- computes baseline-first SSR bootstrap when hints exist
- exposes Vue composables via `@birthday-cake-loading/nuxt/runtime`
