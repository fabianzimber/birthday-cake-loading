# Birthday-Cake Loading — Astro demo

Minimal Astro SSR demo using:
- `@birthday-cake-loading/astro` integration (adds middleware)
- `CakeBootstrap.astro` helper (passes `Astro.locals.bcl.bootstrap` into an island)
- React island that mounts `CakeProvider` with `bootstrap`

## Run

```bash
npm install
npm run dev
```
